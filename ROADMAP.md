# CIOS Roadmap

## Genesis Phase

Goal: Establish the working foundation for CIOS inside GitHub, ChatGPT and Codex.

### Sprint 1 – Repository Bootstrap

- Create GitHub repository.
- Add README and governance files.
- Establish folder structure.
- Add first sample data.
- Prepare Codex for development.

### Sprint 2 – MVP-001 Foundation

- Create opportunity input schema.
- Create sample opportunity file.
- Implement Transformation Pressure scoring.
- Implement first reasoning output format.
- Add basic command-line execution.

### Sprint 3 – Opportunity Intelligence Assistant

- Generate opportunity assessment.
- Generate risks.
- Generate win themes.
- Generate recommended next actions.
- Generate evidence trail.

### Sprint 4 – Memory and Learning

- Store past assessments.
- Capture outcomes.
- Prepare learning feedback loop.
