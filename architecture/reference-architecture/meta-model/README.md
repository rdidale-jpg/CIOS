## Purpose

This folder contains the authoritative conceptual meta model for CIOS Reference Architecture.

RA-001 sits above the Founding Papers, Enterprise Intelligence papers, Reference Architecture and Runtime Architecture. It defines the first-class architectural objects that CIOS may reason about, persist, relate, derive, act upon and learn from.

## Documents

- [RA-001 — CIOS Enterprise Intelligence Meta Model](RA-001-CIOS-Enterprise-Intelligence-Meta-Model.md) — the foundational metamodel for Enterprise Intelligence objects, relationships, inheritance, lifecycle, state, temporality and evidence lineage.

## Rule

Every CIOS runtime, product surface, data model, reasoning model and generated architecture extension must conform to RA-001 or explicitly propose a governed extension to it.
