# CIOS Reference Architecture

**Status:** Draft  
**Owner:** Rob / CIOS  
**Last updated:** 2026-07-03

## Purpose

The CIOS Reference Architecture is the recommended entry point to the full CIOS architecture. It does not replace the Founding Papers or Enterprise Intelligence papers; it organises, explains and connects them so a new human contributor or AI coding agent can understand the project before touching code.

The Reference Architecture defines:

- what CIOS is;
- why CIOS exists;
- how CIOS reasons;
- what CIOS knows about enterprises;
- how Flora operationalises the architecture;
- the common vocabulary;
- the first principles;
- the maturity model;
- the roadmap.
