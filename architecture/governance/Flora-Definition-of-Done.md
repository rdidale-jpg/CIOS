# Flora Definition of Done

**Purpose:** Define completion criteria for Flora feature work without changing application functionality or runtime behaviour.

## Definition of Done

A Flora feature is complete only when:

✓ Code complete

✓ Tests complete

✓ Documentation complete

✓ Architecture review complete

✓ FA-001 compliant

✓ ADR updated if required

✓ UX updated if required

✓ Explainability verified

✓ Lineage verified

✓ Unknowns preserved

✓ Commercial reasoning validated

## Completion Rule

Do not mark Flora feature work complete unless the implementation review records how the feature remains aligned with FA-001, the Flora ADR record, the Product Blueprint, the UX Journey and the Reference Architecture.
