## Purpose

The Commercial Intelligence Playbook Library is a governed CBOK knowledge-asset set for turning commercial signals, pressures, AI reinvention indicators, competitor hypotheses, stakeholder influence and Commercial DNA into actionable recommendations. It creates reusable knowledge, not runtime behaviour.

## Library Contents

| Asset type | Files | Use |

## Governance

These assets shall be maintained as CBOK knowledge assets under CBOK-SCI-001. Patterns and anti-patterns are heuristics unless promoted through evidence. Competitor playbooks are hypotheses and must be validated before use in customer-facing recommendations. Flora consumers must preserve evidence, confidence and traceability.
