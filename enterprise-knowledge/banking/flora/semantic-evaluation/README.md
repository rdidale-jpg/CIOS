# Lloyds Semantic Evaluation Gate

This folder contains the Increment 2 architecture-readiness gate for testing whether Flora can understand Lloyds Banking Group source content rather than merely reading governed object structure.

Files:

- `Content-Understanding-Evaluation.md` — evaluation specification, blind test cases, expected answer key, review rubric, results, failure analysis, corpus adequacy assessment and readiness decision.
- `Lloyds-Semantic-Evaluation-Dataset.json` — Lloyds-specific passages, derived Evidence, Observations, Unknowns, candidate Contradictions, authority and lineage.

Decision: **Not ready — corpus or semantic runtime inadequate** until Mode A/B/C Flora outputs and two-reviewer scorecards are committed.
